## 指南
### 比赛信息
1. 比赛时间:4月中（校赛）；5月（省赛）；
2. 比赛类别：B类
3. 比赛内容：
- 和期末考试考纲相同（校赛）;
- 80%和考纲相同（省赛）;
### 心得1
1. 一个笔试的竞赛。内容基本和大学物理期末的考纲重合。
2. 统计意义上来讲，如果大学物理期末考试在90+,那么同等实力在校赛拿二等奖以上没问题；
3. 一等奖选去参加省赛（五月），《大学物理》课程的考纲覆盖大约在80%左右。这个成绩拿省一应该也是很轻松的事。（听说，未经个人实际验证）
4. 推荐在在大二下的阶段参加。类似于备考一门考试，拿之前的题练练手即可。零碎几天的时间，就可以拿个省一/二；对于应试能力较好的人，参加这个比赛就是顺手牵羊的事。
